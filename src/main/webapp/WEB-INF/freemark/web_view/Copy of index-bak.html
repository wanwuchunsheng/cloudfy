服务内容

UI和网站设计

可根据客户不同需求定制化设计,主要包括：UI设计，网站设计，海报设计，广告设计。

网站开发

从前期网站设计，确定设计样稿。到后期切片，生成静态HTML页面，使用主流DIV+CSS定位完成， 网站兼容市场主浏览器:IE9+、Safari、火狐、谷歌等。

后台管理系统

云朵家后台管理系统V2.0由云朵家团队独立设计开发，该系统使用简单，操作方便，界面美观。方便客户很好管理自己网站信 息内容,系统主要功能：权限管理，用户管理，新闻管理，商品商城管理，图表统计，短信邮件系统等。

小程序应用

可定制化开发 微信小程序，如：KTV，餐饮点餐，电商类（购物）APP等，由云朵家后端管理系统提供PC端服务。 根据小程序缓存机制，我们特定设计一套最佳响应方案，提高小程序响应速度，尽可能降低4G信号差带来的体验困扰。

性能响应

优秀的业务解决方案，能提高软件运行性能，稳定性。云朵家出品软件能保证页面打开响应时间2秒内；微信小程序页面响应时间平均速度300毫秒。
