服务内容

UI和网站设计

可根据客户不同需求定制化设计,主要包括：UI设计，网站设计，海报设计，广告设计。

网站开发

从前期网站设计，确定设计样稿。到后期切片，生成静态HTML页面，使用主流DIV+CSS定位完成， 网站兼容市场主浏览器:IE9+、Safari、火狐、谷歌等。

后台管理系统

云朵家后台管理系统V2.0由云朵家团队独立设计开发，该系统使用简单，操作方便，界面美观。方便客户很好管理自己网站信 息内容“权限管理，用户管理，新闻管理，商品管理，报表统计，短信邮件系统等。

移动应用

可定制开发 微信小程序，如：KTV，餐饮点餐（属于自己的美团外卖APP）等，由云朵家后端管理系统提供PC端服务。

性能响应

根据客户需求，提供性能最佳的设计方案。在网络状态良好情况下，PC端软件页面打开响应时间4秒内；微信小程序页面响应时间平均速度300毫秒。

售后服务
